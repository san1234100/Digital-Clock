# Digital Clock Project

This is a simple digital clock web application built with HTML, CSS, and JavaScript.

## Demo

[Digital Clock Demo](https://san1234100.github.io/Digital-Clock/)

## Features

- Real-time digital clock display
- Simple and clean user interface

## Installation

1. Clone the repository:

    ```bash
    https://github.com/san1234100/Digital-Clock.git
    ```

2. Open the project folder:

    ```bash
    cd Digital-Clock
    ```

3. Open the `index.html` file in your web browser.

## Usage

1. Open the `index.html` file in your web browser.
2. The digital clock will be displayed, showing the current time.
